I create useful web applications with code and user design.

Blocitoff

Project for Bloc's Front-End Development Program

Self-destructing task list that automatically deletes tasks if not completed within 7 days. Users create individual accounts by providing a username, email address, and password. They can then create tasks and delete them when finished. This app was built using AngularJS, Express, Mongoose, HTML5, and LESS (CSS3) with Bootstrap.

Bloc-Pomodoro

Project for Bloc's Front-End Development Program

Web application for using the Pomodoro technique. It includes 3 timers: Pomodoro, Short Break, and Long Break. There is also task functionality, in which users can create and delete tasks for the pomodoro sessions. The app was built using AngularJS, Express, Mongoose, HTML5, and LESS (CSS3) with Bootstrap.

Listy

Project for Bloc's Front-End Development Program

Users can create an account in which they make and edit bookmarks. The bookmarks can include a link, comments, and tags. Users can search bookmarks and sort them by tags. The tags can be edited and deleted. The app was built using AngularJS, Express, Mongoose, HTML5, and LESS (CSS3) with Bootstrap.
