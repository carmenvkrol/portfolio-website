Blocitoff

Project for Bloc's Front-End Development Program

I built a self-destructing task list which automatically deletes tasks if not completed within 7 days. Users create individual accounts by providing a username, email address, and password. They can then create tasks and delete them when finished. The app was built using AngularJS, Express, Mongoose, HTML5, and LESS (CSS3) with Bootstrap.

Narwhal Skewers

The successful Kickstarter project, Narwhal Skewers, needed a product website to show potential clients. In order to meet a tight deadline and facilitate updates by the owner, I modified the CSS, HTML, and PHP of a WordPress template.
